# ToolBird

A Minecraft mod for version 26.1.1 built with Fabric.

## Features

### Tool Leveling
Tools gain Efficiency enchantments automatically as you use them.
Mine 20 blocks for Efficiency I, 50 for Efficiency II, 100 for Efficiency III.

### Bridge Maker
Right-click on any block to instantly place a 5-block oak plank
bridge in front of you. Perfect for crossing gaps quickly.

![Bridge_Maker_Recipe](others/Bridge%20block%20craft.png)

### Bird Feather
Right-click to launch yourself into the air for a bird's-eye view
of your surroundings. Floats you safely back down. 15 second cooldown.

![Bird_Feather_Recipe](others/Wings%20craft.png)

## Installation
1. Install [Fabric Loader](https://fabricmc.net/use/installer/) for Minecraft 26.1.1
2. Download [Fabric API](https://modrinth.com/mod/fabric-api)
3. Drop the ToolBird `.jar` into your `mods` folder
